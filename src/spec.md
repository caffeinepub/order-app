# Specification

## Summary
**Goal:** Remove four specific UI elements from the Settings screen without affecting any other UI.

**Planned changes:**
- Remove the Settings card header/title area at the specified XPath.
- Remove the first Settings input at the specified XPath.
- Remove the first Settings label at the specified XPath.
- Remove the second Settings label at the specified XPath.

**User-visible outcome:** The Settings page no longer shows the selected header/title area and the specified input/labels; all other Settings UI remains unchanged and English text stays in English.
